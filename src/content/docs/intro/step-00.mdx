### Mari kita mulai

ReactJS sebenarnya sudah cukup keren untuk membuat aplikasi web modern. Namun, ada beberapa hal yang perlu kita lakukan secara manual, seperti:

- Deklarasi Routing masih harus menggunakan library tambahan
- Masih sulitnya untuk membuat aplikasi yang SEO friendly
- Ada permasalahan dengan performa aplikasi
- Untuk membuat BackEnd, kita harus membuatnya sendiri

Nah bagaimana nih bila seandainya ReactJS itu memiliki fitur-fitur tersebut secara out-of-the-box? 

Nah, **NextJS** adalah (salah satu) jawabannya.

### NextJS itu Apa?

NextJS adalah framework ReactJS yang dibuat oleh Vercel. NextJS memberikan fitur-fitur yang dibutuhkan oleh aplikasi web modern, seperti:

- Routing yang mudah
- SEO friendly
- Performa yang cepat
- Pembuatan Routing yang bisa menghasilkan API dengan mudah

Dalam perkembangannya NextJS, framework ini terpecah menjadi 2 versi, yaitu:

- Page Router
- App Router

`Page Router` adalah versi NextJS yang kita kenal sebelumnya. Dimana kita membuat routing berdasarkan file yang ada di folder `pages`.

Sedangkan `App Router` adalah versi NextJS yang lebih baru. Dimana kita membuat routing berdasarkan file yang ada di folder `app`.

Nah, dalam tutorial ini kita akan menggunakan versi App Router.

Jadi yuk, tanpa _bak bik buk_, kita mulai tutorialnya.
