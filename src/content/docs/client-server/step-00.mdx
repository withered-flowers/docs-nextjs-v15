### Harapan Pembelajaran

Setelah bagian ini selesai, situ diharapkan dapat:

- Mengerti tentang perbedaan `Client Component` dan `Server Component`
- Mengerti cara Opting-In / Out dari `Data Caching`
- Mengerti tentang cara penggunaan `Server Actions`

### Clone Aplikasi

```sh
git clone https://github.com/withered-flowers/sources-nextjs-client-server-component.git
```

### Menyalakan Server

1. cd `/sources/a-start/server`
1. install package yang dibutuhkan dengan `npm install`
1. jalankan server dengan `npm run watch`

### Menyalakan client

1. cd `/sources/a-start/client-15`
1. install package yang dibutuhkan dengan `npm install`
1. jalankan client dengan `npm run dev`
