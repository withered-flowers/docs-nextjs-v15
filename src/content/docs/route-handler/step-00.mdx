### Harapan Pembelajaran

Setelah bagian ini selesai, situ diharapkan dapat mengerti tentang:

- Route Handler
- Zod (Schema Validation)

### Clone Aplikasi

```sh
git clone https://github.com/withered-flowers/sources-nextjs-route-handler.git
```

### Disclaimer

- Pembelajaran ini akan menggunakan `drizzle` sebagai ORM dan `sqlite` untuk databasenya, tepatnya dengan `libSQL`.

  :::caution
  Semoga bisa diterima yah, kl ga bisa diterima, mungkin situ bisa coba baca baca dulu, siapa tau tertarik.

  - Untuk tahu lebih lanjut tentang `drizzle` bisa baca [di sini yah](https://orm.drizzle.team/docs/overview)
  - Kalau untuk sqlite-nya, bisa dibaca [di sini yah](https://www.sqlite.org/index.html)

  :::

  :::note
  Kenapa `libSQL` dan bukan `better-sqlite3` ?

  - Biar lebih dekat dengan yang kekinian saja, si `turso`.
  - `turso`, apaan tuh? bisa dibaca [di sini yah](https://docs.turso.tech/introduction)

  :::

- Pembelajaran ini belum sampai tahap melakukan `Authentication`, hanya baru sampai membuat kerangkanya saja.

### Menyalakan Server

1. cd `/sources/a-start/server`
1. install package yang dibutuhkan dengan `npm install`
1. jalankan server dengan `npm run watch`

### Menyalakan client

1. cd `/sources/a-start/client-15`
1. install package yang dibutuhkan dengan `npm install`
1. jalankan client dengan `npm run dev`
